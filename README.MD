# Python Interview Practice 🐍

This repository contains my Python coding practice from **basic to advanced level** to improve problem-solving skills and prepare for technical interviews.

## 🎯 Goal

To become confident in writing Python logic during interviews by practicing daily coding questions and real-world programming problems.

## 📚 Topics Covered

### Basic Python
- Variables and Data Types
- Input / Output
- Conditions (`if`, `else`)
- Loops (`for`, `while`)
- Strings
- Lists
- Tuples
- Sets
- Dictionaries

### Intermediate Python
- Functions
- List Comprehensions
- Lambda Functions
- Exception Handling
- File Handling
- Modules
- Recursion
- Pattern Programs

### Advanced Python
- Object-Oriented Programming (OOP)
- Inheritance
- Polymorphism
- Decorators
- Iterators / Generators
- Collections Module
- Problem Solving
